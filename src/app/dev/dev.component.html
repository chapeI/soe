the goal is to build a simple scheduler with the following aims in mind
  • easy to follow code code
    • good naming convention
    • refactor into functions
  • simple UI
    • no course codes
    • no weights (ie. 2 courses from math to complete this requirement)
    • simplified requirement headings
  • powered by
    • open source code
    • simple json to hold data
    • firestore document updates courses
  • respond to screen size changes with FlexBox
  • persistence
  • dark mode
