about

Scheduler is built for anyone curious about obtaining a degree in software. To build their own schedule or follow one from an institution.
  • view requirements from different schools
  • what kind of courses will I be taking if I attend this institution?
  • follow a cirriculum or build your own

design

Simplicity was always in mind in the UI and in writing code
  • simplest possible UI
    • no course codes
    • general heading names
    • no weights (ie. 2 courses from math to complete this requirement)
  • load data with firestore
    • convert a JSON object into a firestore doc
    • open source for pull requests
  • persist without having to create an account
  • dark mode
  • responsive to screen size changes
