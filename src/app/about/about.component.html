design

Simplicity for the UI and writing code
  • simplest possible UI
    • no course codes
    • general heading names
    • no weights (ie. 2 courses from math to complete this requirement)
  • load data with firestore
    • convert a JSON object into a firestore doc
    • open source for pull requests
  • persist without having to create an account
  • dark mode
  • responsive to screen size changes
  • writing simple code is not simple
    • good naming conventions
